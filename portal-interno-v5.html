Cada panel funciona como entrada a un módulo de información. Los enlaces reales a servicios de nube privada deben protegerse con autenticación, VPN o red institucional.

Aviso importante: este portal es una maqueta estática. No publicar en GitHub Pages credenciales, IP internas, tokens, bases crudas, evidencias reservadas ni rutas sensibles. Los enlaces mostrados son de ejemplo o estado pendiente hasta definir URLs institucionales protegidas.

Nube Privada FOSS HomeEnergy

Infraestructura para adquisición, almacenamiento, visualización y análisis de datos del proyecto.

Volver a paneles

Arquitectura funcional

Sensores IoT→ Node-RED→ InfluxDB→ Grafana/ JupyterLab→ Reportes
Adquisición IoT

Node-RED

Gestiona flujos de recepción, transformación y envío de datos desde refrigeradores instrumentados.

  • Recepción de datos de sensores.
  • Validación básica y transformación.
  • Envío hacia InfluxDB.
  • Alertas o reglas simples.
Acceso pendiente Requiere red interna o VPN.
Series temporales

InfluxDB

Almacena variables térmicas, eléctricas, ambientales y operativas como series temporales.

  • Temperatura interna y ambiente.
  • Potencia, corriente y energía.
  • Aperturas, ciclos y eventos.
  • Datos crudos y procesados.
Acceso restringido No publicar URL ni token.
Visualización

Grafana

Muestra dashboards públicos resumidos y tableros técnicos internos para seguimiento energético.

  • Consumo energético diario.
  • Temperaturas y humedad.
  • Ciclos de compresor.
  • Comparación entre refrigeradores.
Dashboard técnico Dashboard público
Análisis

JupyterLab

Permite análisis exploratorio, limpieza, modelado, gráficas, notebooks y generación de reportes.

  • Notebooks de análisis.
  • Limpieza y validación de datos.
  • Gráficas y modelos.
  • Exportación de resultados.
